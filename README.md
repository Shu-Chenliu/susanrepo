# susanrepo
把檔案輸入github 的方法
1. 打開git bash 跟 github
2. 在github 創一個repo
3. 在git bash 打 cd 然後貼上要傳的檔案的位置 如: C:\Users\susan\OneDrive\桌面\程式\homework\src\homework 注意: 斜線要改方向
4. 複製ssh
5. 在git bash 打 git clone 然後貼上ssh連結
6. 用vscode 打開那個資料夾
7. 在vscode 按commit
8. 去git bash 看git status(應該是紅色的)
9. 打git add
